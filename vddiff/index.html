vddiff—Text Terminal Directory Diff Tool

vddiff is a simple text terminal directory diff tool using vim -dR for the actual diff operation (other diff tools can be configured). A file browser with lynx and vi like navigation displays the names found in the directories given on the command line:

Opening a file with differences startes vim in diff mode (if vim is not installed, “diff file1 file2 | less” is used instead):

Merging of file contents is done with vim or the selected diff tool. If the available diff tools don't support merging, files can be opened separately in the file browser to edit them with e.g. vi. File system level merging (copying, renaming or deleting files or directories) can be done with the tool itself.

See the man page for the complete documentation. Releases and download and installation instructions are found on the project source code page.
