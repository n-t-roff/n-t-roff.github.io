vddiff—Text Terminal Directory Diff Tool

vddiff is a simple text terminal directory diff tool. It compares two directories and uses vim -dR for the actual file diff operation (other diff tools can be configured). A file browser with lynx and vi like navigation displays the names found in the directories given on the command line:

Opening a file with differences startes vim in diff mode (if vim is not installed, “diff file1 file2 | less” is used instead):

Merging of file contents is done with vim or the selected diff tool. If the available diff tools don't support merging, files can be opened separately in the file browser to edit them with e.g. vi. File system level merging (copying, renaming and deleting files or directories) is done with the tool itself.

Features:

  • Transparent comparison of compressed files and directories.
  • By use of marks files and directories can be compared which had been renamed or compressed in one of the file trees, and of of files which are in different directory levels.
  • In contrast to diff(1) symbolic links are not followed by default.
  • All used build tools and library functions are POSIX. The tool had been tested on NetBSD, OpenBSD, FreeBSD, Linux, and Solaris.
