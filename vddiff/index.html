vddiff—Text Terminal Directory Diff Tool

vddiff is a simple text terminal directory diff tool using vim for the actual diff operation. A file browser with lynx like navigation displays the names found in the directories given on the command line:

Opening a file with differences startes vim in diff mode:

See the man page for a short documentation. vddiff can be downloaded at https://github.com/n-t-roff/vddiff. It is recommended (but not required) to install the speed optimized AVL library libavlbst before the software is build and installed with:

  
  $ ./configure
  $ make
  $ su
  # make install
  # exit
  $ make distclean
